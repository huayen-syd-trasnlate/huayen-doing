普贤承华严宗

三大法脉四大传承源流

一，普贤乘华严宗的传承是从如来 (毗卢遮那如来〈法身〉) → 普贤菩萨〈形而下本体界〉 → 释迦牟尼佛〈化身〉 → 大龙菩萨 → 龙树菩萨 → 马鸣菩萨 → 实叉难陀三藏法师 → 杜顺祖师 → 智俨祖师 → 贤首祖师…… → 敬缘钦因 (四十一世祖师) → 承观世音菩萨嘱咐传四十二世昌一继梦

二、普贤乘华严宗的大乘瑜伽行法传承是从梵 → 伊西瓦 → 希瓦 → 拉克鲁西 (Dadagi) → 普拉梵纳德 (Swami Pranavanand) → 克利普梵纳德 (Swami Kripalvanandji) → 惹查西牟尼 (Swami Rajarshi Muni) → 承普贤菩萨嘱托至美国寻找 → 传法于昌一继梦

三、普贤乘华严宗的禅宗传承是从菩提达摩 → 慧可 → 僧灿 → 道信 (建立东山法门) → 弘忍 (东山法门第一代祖师) → 惠能 → 南狱怀让 → 马祖道一 → 百丈怀海 → 黄蘗希运 → 临济义玄 (下开临济宗第一代祖师) → ……

备注:

临济下第二十五世碧峯，第七世突空智板禅师，演派十六字:
智慧清净 道德圆明 真如性海 寂照普通

至五台、峨嵋、普陀前寺，续演派三十二个字:

心源广续 本觉昌隆 能仁圣果 常演宽宏

惟传法印 证悟会融 坚持戒定 永继祖宗

本宗至北京海定区药王寺修林和上传觉醒，觉醒又号梦参，梦参承地藏菩萨付托传昌一继梦(昌一法名为地藏菩萨所取，梦参取名曰继梦，继梦自号海云)，下传隆字辈为临济宗第四十八世。

四、华严义学另出新法脉: 此法脉乃清末民初佛教知识青年有感于国难当头，积极于华严经中寻找淑世法门，故而入终南山闭关，故有终南五虎将之称，于中有显珠月霞尊者开华严大学广收学子，有普海慈舟尊者传觉醒梦参祖师，再传昌一继梦。

是故普贤乘华严宗有此四大传承。
